How to use the scripts: Rename one of the TXT files to LUA (Note: don't run all 4 scripts at the same time, they will cause severe conflicts).

1. OpponentChase: Only player's icon was being moved to the end of the healthbar (right)
2. OpponentChaseFlip: Same as "OpponentChase", but the healthbar was being flipped
3. PlayerRun: Only opponent's icon was being moved to the end of the healthbar (left)
4. PlayerRunFlip: Same as "PlayerRun", but the healthbar was being flipped
