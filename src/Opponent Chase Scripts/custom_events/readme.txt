How to use "OpponentChaseEvent" and "PlayerRunEvent":
1. First, in the chart editor, put one of the custom events
2. Then, you choose one of Value 1 or Value 2:
- OpponentChaseEvent:
+ If Value 1 = true/false, it'll activate like OpponentChase/deactivate and restore default
+ If Value 2 = true/false, it'll activate like OpponentChaseFlip/deactivate and restore default
- PlayerRunEvent:
+ If Value 1 = true/false, it'll activate like PlayerRun/deactivate and restore default
+ If Value 2 = true/false, it'll activate like PlayerRunFlip/deactivate and restore default

Note: Please choose one of the custom events and values, don't activate all events values, it'll conflicts
